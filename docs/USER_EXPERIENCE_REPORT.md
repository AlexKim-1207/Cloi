2. **사용자 통찰을 무시하지 말 것** — 사용자가 "성별/가격 soft signal" 명확히 짚어줬는데 production은 그것 0% 반영.
3. **유저 페르소나 측정은 분기마다 의무화** — 매주/매분기 5케이스 재측정 → 회귀 즉시 감지.
4. **응답 강건성도 지표** — 5케이스 중 2개 실패 (40%) — 정확도 못지않게 중요.
5. **단순 케이스 ≠ 복잡 케이스** — T1처럼 단순 케이스만 잘 되는 시스템은 실유저 만족 어려움.
