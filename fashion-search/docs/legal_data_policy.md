# 데이터 법적 정책

## 금지 사항

1. **경쟁사 앱 크롤링 학습 금지** — 무신사, 에이블리, 지그재그 등 경쟁 플랫폼 이미지/데이터를 동의 없이 수집 후 모델 학습 데이터로 사용 금지.

2. **유저 업로드 이미지 학습셋 전환 금지** — 사용자가 검색 목적으로 업로드한 이미지를 동의 없이 모델 학습에 사용 금지. 사용 동의 획득 시 별도 명시적 옵트인 필요.

3. **DeepFashion 상업 라이선스 확인** — DeepFashion 데이터셋은 비상업적 연구 목적으로만 배포됨. 상업 서비스 학습 코어로 사용 시 별도 계약 필요.

4. **개인정보 포함 이미지 처리** — 얼굴, 개인 식별 정보가 포함된 이미지는 즉시 삭제. 검색 완료 후 업로드 이미지 서버 보관 금지.

## 카탈로그 데이터 정책

- 브랜드/판매사와 데이터 제공 계약 필수 (`src/data/contracts/`)
- 계약 없는 상품 이미지 카탈로그 추가 금지
- 권리 매트릭스 (`src/data/rights/rights_matrix.csv`) 주기적 업데이트

## 캐시 정책

- 검색 결과 캐시 TTL: 24시간
- 캐시 만료 후 자동 삭제
- 개인 식별 가능한 정보는 캐시에 저장하지 않음
