다음 용도로의 사용은 금지:
- 외부 서비스 판매 또는 재배포
- 모델 학습 데이터 (별도 동의 필요)
- 경쟁사 서비스 제공

### 데이터 관리

- 계약 종료 시 30일 내 데이터 삭제
- 보안 사고 발생 시 24시간 내 통보
- 연 1회 데이터 감사 허용
